## SimpleAuth - Make your server safer
**SimpleAuth** - minecraft plugin that will allow you to protect your players' accounts!

## Features:
- Easy to use and easy to set up;
- Protection by **IP**;
- Automatic Sessions;

## Commands:
```/login```, ```/reg```
